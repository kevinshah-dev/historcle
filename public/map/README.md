# World Map Data

`land-110m.json` is extracted from `world-atlas@2.0.2`, a prebuilt TopoJSON redistribution of Natural Earth vector data.

- Package: https://github.com/topojson/world-atlas
- Source data: https://www.naturalearthdata.com/
- Package license: ISC
- Natural Earth data: public domain

The app projects the unprojected spherical coordinates in this file into a local SVG map for the Historcle location picker.
