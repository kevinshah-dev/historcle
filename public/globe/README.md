# Globe Textures

These compact Earth texture assets are served locally for the Historcle WebGL globe:

- `earth_atmos_2048.jpg`
- `earth_clouds_1024.png`

Source: Three.js example planet textures from `mrdoob/three.js`, downloaded from:

- https://raw.githubusercontent.com/mrdoob/three.js/dev/examples/textures/planets/earth_atmos_2048.jpg
- https://raw.githubusercontent.com/mrdoob/three.js/dev/examples/textures/planets/earth_clouds_1024.png

They are intentionally 2K/1K rather than 8K so the game stays responsive on mobile.
